### Запуск:
#### Установка:
**Клон репозитория**

```bash
git clone https://github.com/yinmus/clp.git
cd clp
```

**Установка зависмостей**:

`Простая Компиляция`:

```bash
make
```
`Компляция, перенос бинарника в ~/.local/bin/`

```bash
make install
```

`Удалить`

```bash
make uninstall
```

`Очистка`

```bash
make clean
```
