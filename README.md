# clp

## Легкий, консольный аудио плеер

### Форматы:
- `MP3`
- `WAV`
- `OGG`

### Зависимости:

- `SDL2` (libsdl2-dev) — библиотека для работы с мультимедиа.
- `SDL2_mixer` (libsdl2-mixer-dev) — расширение для воспроизведения аудиофайлов.
